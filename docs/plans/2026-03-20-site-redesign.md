# Empire 8 Site Redesign — Cinematic Frontend

> **Reference:** terminal-industries.com (scroll animations, video backgrounds, parallax, cinematic sections)
> **Brand assets:** Header splash PSD, Interior base PSD, 20-page sales deck

**Goal:** Rebuild empire8ny.com frontend into a cinematic, animation-heavy experience with scroll-triggered reveals, parallax, counter animations, and video backgrounds while preserving all backend functionality.

**Architecture:** Install GSAP + ScrollTrigger + Lenis for smooth scroll and cinematic animations. Rebuild homepage and key pages section-by-section. Use the brand PSD assets as hero/section backgrounds. Dark theme throughout.

**Tech:** Next.js 16, Tailwind v4, GSAP + ScrollTrigger, Lenis smooth scroll
